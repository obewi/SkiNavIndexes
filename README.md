SkiNavIndexes

A minimal Python package scaffold for building ski navigation indexes.

Quick start
- Ensure you have Python installed.
- Use the uv package manager to install dependencies (see memory note below).
- Create a virtual environment and install requirements as needed.
- Run basic index building with the provided Python module.

Project structure
- src/ski_nav_indexes/__init__.py
- src/ski_nav_indexes/index.py
- .gitignore
